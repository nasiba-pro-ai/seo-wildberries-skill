# SEO Wildberries для Codex

Навык собирает и очищает семантическое ядро, анализирует конкурентов Wildberries через встроенный браузер Codex и формирует итоговый Excel.

## Установка

### Через ссылку на репозиторий

Откройте Codex и напишите:

`Используй $skill-installer и установи навык по этой ссылке: https://github.com/nasiba-pro-ai/seo-wildberries-skill`

После установки перезапустите Codex, если навык не появился сразу.

### Вручную из ZIP

1. Распакуйте архив.
2. Переместите папку `skill-seo` в папку навыков пользователя:
   - macOS: ~/.codex/skills/skill-seo
   - Windows: %USERPROFILE%\.codex\skills\skill-seo
3. Перезапустите Codex.
4. Откройте `Плагины -> Навыки` и найдите `SEO Wildberries`.

## Первый запуск

Создайте новую задачу и напишите:

`Используй $skill-seo и собери SEO для моей карточки Wildberries.`

Навык запросит характеристики товара, основной поисковый запрос и минимальную выручку конкурента.

## Важно

- Вход в Wildberries и Wildbox выполняйте самостоятельно. Никому не передавайте пароль.
- Навык не публикует карточку и не меняет кабинет продавца.
- Проверяйте итоговый текст и характеристики перед загрузкой на Wildberries.
