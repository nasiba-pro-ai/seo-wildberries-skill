### Вручную из ZIP

1. [Скачайте SEO-навык в ZIP](https://github.com/nasiba-pro-ai/seo-wildberries-skill/archive/refs/heads/main.zip).
2. Откройте папку «Загрузки» и распакуйте архив.
3. Переименуйте распакованную папку `seo-wildberries-skill-main` в `skill-seo`.
4. Переместите папку `skill-seo` в папку навыков:

- macOS: `~/.codex/skills/skill-seo`
- Windows: `%USERPROFILE%\.codex\skills\skill-seo`

5. Перезапустите Codex.
6. Откройте **Плагины → Навыки** и найдите **SEO Wildberries**.
