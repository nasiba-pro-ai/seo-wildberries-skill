# Правила контента Wildberries

Источник правил: официальные материалы Wildberries `Как создать карточку товара` и `Выдача и ранжирование товаров`, сохраненные пользователем 11.07.2026.

## Заголовок

- Максимум 60 символов.
- Коротко и точно отвечать, что изображено на фото.
- Не использовать синонимы и повторы слов.
- Не добавлять неподтвержденные свойства и лишние подробности.
- Не перечислять несколько названий через запятую или слэш.
- Не указывать пол, возраст, бренд и сезон.
- Не использовать оценочные слова, заглавные фразы, эмодзи и спецсимволы.
- Для этого скилла использовать схему: `тип товара + фасон/свойство + цвет`.

## Характеристики

- Заполнять только точными значениями реального товара.
- Заполнять все значимые доступные поля категории.
- Выбирать предлагаемые Wildberries значения, когда они доступны.
- Несколько значений вносить отдельно, а не строкой через запятую.
- Не дублировать одно значение в разных полях.
- Не добавлять прочерки, лишние надписи и SEO-фразы вместо значения.

## Описание

- Писать красиво, убедительно, правдиво и по существу.
- Целевой объем — 900–1500 символов. Если кабинет Wildberries показывает меньший лимит для категории, соблюдать лимит категории.
- Использовать 8–15 наиболее релевантных высоко- и среднечастотных запросов из очищенного ядра.
- Встраивать ключи естественно в предложения; точное вхождение использовать только когда оно не ломает речь.
- Каждый ключ или близкую словоформу по возможности использовать один раз.
- Рассказать об особенностях, преимуществах и сценариях использования.
- Не перечислять характеристики, которые имеют отдельные поля.
- Не вставлять теги или список SEO-запросов.
- Не указывать чужие бренды, домены, артикулы, спецсимволы и нерелевантные варианты товара.
- Не повторять ключевые слова и их формы.
- Включать необходимые слова естественно, с пробелами и без переспама.
- Учитывать лимит описания конкретной родительской категории, который показывает кабинет Wildberries.
- Не придумывать выгоды, свойства, состав, назначение или комплектацию, которых нет в ТЗ.
- Не использовать пустые рекламные штампы: «лучший», «идеальный для всех», «хит», «премиум качество», если они не подтверждены.

## Структура описания

1. Первый абзац: что это за товар и какой образ он создает.
2. Второй абзац: фасон, посадка, материал и подтвержденные особенности.
3. Третий абзац: практические преимущества для покупателя.
4. Четвертый абзац: сценарии использования и сезонность.
5. Пятый абзац: сочетания с другими предметами.
6. Финальная фраза: комплектация и важные уточнения без давления на покупателя.

## Ограничение аналитики

Wildberries указывает, что сторонние сервисы оценивают продажи приблизительно. Использовать расчетную выручку Wildbox как критерий отбора по решению пользователя, но помечать ее как оценочную, а не как официальные данные Wildberries.
